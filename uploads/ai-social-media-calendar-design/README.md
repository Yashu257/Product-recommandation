# PostAI - AI-Powered Social Media Calendar

A modern, Buffer-inspired SaaS platform for managing social media content with AI-powered features.

## Features

### 📊 Dashboard
- Real-time statistics (Total Posts, Scheduled, Drafts, Published)
- Upcoming schedule overview
- Recent posts timeline
- AI content suggestions
- Quick action buttons
- Content performance metrics

### 📅 Social Calendar
- **Month View**: Full calendar grid with color-coded posts
- **Week View**: Detailed weekly schedule
- Status indicators (Draft, Scheduled, Published)
- Click posts to view details
- Navigate between months/weeks

### ✍️ Create Post
- Title and content editor
- Multi-platform selection (Twitter/X, Facebook, Instagram, LinkedIn)
- Date and time scheduling
- AI-powered features:
  - Generate Caption
  - Generate Hashtags
  - Rewrite Content
  - Improve Engagement
- Live preview panel with platform-specific character limits
- Save as draft or schedule

### 📄 Post Details
- Full post content view
- Platform indicators
- Hashtag display
- Schedule information
- Actions: Edit, Delete, Duplicate, Reschedule

### 🤖 AI Assistant
- Interactive AI chat interface
- Quick action suggestions
- Content ideation
- Engagement optimization
- Smart hashtag generation

### ⚙️ Settings
- Profile settings
- Notifications
- Privacy & Security
- Billing
- Appearance
- Connected accounts

## Design Features

- Clean white background with soft pastels (purple, pink, blue)
- Large rounded cards (rounded-2xl)
- Minimal shadows with hover effects
- Spacious layouts with generous padding
- Smooth micro-interactions
- Modern typography
- Responsive desktop-first design
- Buffer-inspired aesthetics with AI-first approach

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS 4
- Vite
- Lucide React (icons)
- date-fns (date manipulation)

## Color Palette

- Primary: Purple (#8B5CF6) to Pink (#EC4899) gradients
- Secondary: Blue, Green, Orange accents
- Neutral: Gray scale for text and backgrounds
- Status Colors:
  - Draft: Gray
  - Scheduled: Purple
  - Published: Green
